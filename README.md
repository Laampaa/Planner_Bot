# Reminder Bot

Это Telegram-бот для создания напоминаний, которые отправляются в Telegram-канал. Бот анализирует текстовые и аудио сообщения, извлекает задачу и дату/время исполнения, а затем создает напоминание в канале.

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/yourusername/reminder-bot.git
   cd reminder-bot

## ⚠️ ВАЖНО: добавьте бота в канал

Перед использованием бота необходимо:

1. Добавить бота в нужный Telegram-канал
2. Назначить бота **администратором**
3. Выдать право **«Публиковать сообщения»**
4. Указать `CHANNEL_ID` этого канала в файле `.env`

Пример:
CHANNEL_ID=-1001234567890

Если бот не добавлен в канал или указан неверный `CHANNEL_ID`,
он не сможет отправлять напоминания.
